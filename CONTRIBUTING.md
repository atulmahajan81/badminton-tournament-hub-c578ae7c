# Contributing to Badminton Tournament Hub

Thank you for considering contributing to the Badminton Tournament Hub! Here is how you can help:

## Pull Request Process

1. Fork the repository and create your branch from `main`.
2. Ensure the code compiles and all tests pass.
3. Follow the coding style outlined in the project.
4. Open a pull request with a clear description of your changes.

## Code Style

- Use PEP 8 for Python code.
- Follow standard ESLint rules for TypeScript.

## Testing Requirements

- All new features must be covered by unit tests.
- Ensure existing tests pass by running `pytest` and `npm test`.
